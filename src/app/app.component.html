FootyVerse Dev Server - Alpha Build

Football Management Sim - Open Source Development

Focus is open source development of a soccer management simulation allowing management of individual players, teams, and also the league. A mixture of light RPG elements and heavy futbol management elements.

The infrastructure is built on Google Cloud running on a Node JS / Angular 5 stack for ease of scalabilty and growth. I am making the project fully open source (which is a first for me).

My goal with this project is to challenge myself, enjoy honing my development, and hopefully by making it open-source learn from like minded developers along the way.
